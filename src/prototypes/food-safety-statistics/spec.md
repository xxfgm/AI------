# 食品安全违法案件统计表（按案值、产品分类）

## 📋 业务与功能

### 1.1 核心目标
展示食品安全违法案件的统计数据，按案件价值和产品分类进行详细划分，为监管工作提供数据支持。

### 1.2 功能清单
- **多维度统计**：支持按案值大小和产品类别（如乳制品、脂肪制品、冷冻饮品等）进行分类统计。
- **分级展示**：表格支持多级表头和侧边分类标题（侧向合并单元格）。
- **查询与筛选**：集成办案机构选择、日期范围选择和报表周期选择（月报/季报/年报）。
- **交互功能**：点击案件数量等统计值可查看明细（原型级别实现）。

### 1.3 交互要点
- **侧边分类标题**：侧边“按案值划分”和“食”等分类标题通过纵向合并单元格实现。
- **数值关联**：统计数值（如 3770, 1164 等）以蓝色文字显示，表示可点击查看详情。
- **搜索/导出**：顶部提供标准搜索和导出操作。

## 📊 内容规划

### 2.1 信息架构
```
食品安全违法案件统计表
├── 筛选栏 (办案机构, 日期范围, 报告类型)
└── 统计表格
    ├── 表头 (指标名称, 代码, 案件总数, 食品案件, 特殊食品案件, 案值, ...)
    └── 数据行
        ├── 案件总数
        ├── 免于处罚案件
        ├── 按案值划分 (5万以下, 5-20万, ...)
        └── 食品分类 (小计, 乳制品, 脂肪制品, ...)
```

### 2.2 数据来源
- **数据类型**：统计列表
- **数据源**：生成示例数据（基于用户提供的图片内容）
- **关键字段**：
  - `indicatorName`: 指标名称
  - `code`: 代码
  - `totalCases`: 案件总数
  - `normalProcedure`: 普通程序
  - `foodCases`: 食品案件
  - `specialFoodCases`: 特殊食品案件
  - `caseValue`: 案值 (万元)
  - `fineAmount`: 罚款金额 (万元)
  - `confiscatedAmount`: 没收金额 (万元)
