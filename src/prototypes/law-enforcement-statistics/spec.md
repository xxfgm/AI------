# 执法稽查办案统计表

## 📋 业务与功能

### 1.1 核心目标
> 提供综合执法稽查案件统计的数据展示，帮助管理层直观了解各类型案件的办理情况、罚没金额等关键指标。

### 1.2 功能清单
- **数据筛选查询**：支持按办案机构、处罚决定日期、统计周期（月报/季报/年报等）进行多维度混合查询。
- **数据汇总聚合**：自动合计案件总数、案值、罚款金额等关键数值。
- **多维度表格展示**：分门别类显示各类型案件（标准化、认证认可、计量等）及普通程序的案件数。

### 1.3 交互要点
- 点击「查询」按钮：根据当前表单条件重新请求或筛选展示数据。
- 日期范围与报表类型联动：理论上单选（月报、季报、年报）作为快捷选项，可填充或限制日期选择范围。

## 📊 内容规划

### 2.1 信息架构

```
执法稽查办案统计表
├── 顶部查询区
│   ├── 办案机构 (Select 下拉单选)
│   ├── 处罚决定日期 (DateRangePicker 范围选择)
│   ├── 报表类型快捷键 (Radio 组：月报/一季报/二季报等)
│   └── 操作按钮组 (查询、导出)
└── 主体表格区
    ├── 表头定义 (多级/复杂表头)
    └── 表格数据体 (支持分类汇总、缩进表示层级)
```

### 2.2 数据来源

- **数据类型**：统计报表
- **数据源**：静态 Mock 数据示例（根据图片1内容还原结构）
- **关键字段**：
  - `indicatorName`: 指标名称 (如甲, 合计, 标准化违法案件等)
  - `code`: 代码 (如乙, 1, 2)
  - `totalCases`: 案件总数（件）
  - `normalProcedureCases`: （嵌套/分拆）其中：普通程序
  - `caseValue`: 案值（万元）
  - `fineAmount`: 罚款金额（万元）
  - `confiscatedAmount`: 没收金额（万元）
  - `transferredCases`: 移送司法机关案件（件）

### 2.3 内容示例

- **指标名称**：合计、一、标准化违法案件、二、认证认可与检验检测违法案件...
- **普通程序**：在“案件总数”下作为拆分或独立列展现。

## 🎨 布局与结构

### 3.1 整体布局

- **布局模式**：单栏全宽布局
- **容器宽度**：自适应容器（流式），内容区域铺满视图或固定边距
- **关键尺寸**：
  - 顶部查询区：上下留白 `padding-y: 16px`，元素居中对齐，支持自动折行
  - 表格区：高度自适应填满剩余空间，超出可滚动

### 3.2 响应式适配

- **桌面端（≥1200px）**：完整展示各列字段及完整的筛选栏。
- **平板端/移动端（<1200px）**：支持表格横向滚动；查询条件自适应换行。
